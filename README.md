# Content Moderation Dashboard

##  Problem Statement
Monitoring large volumes of content requires clear insights into harmful patterns and trends.

##  Objective
- Analyze content data
- Identify trends in harmful vs safe content

##  Tools
- SQL
- Excel / Power BI

##  Key Metrics
- % of flagged content
- Category-wise violations
- Time-based trends

##  Status
Dashboard design and dataset preparation in progress.
